## วัตถุประสงค์ของงาน

- ติดตั้งและตั้งค่าสภาพแวดล้อมสำหรับการพัฒนาโปรแกรม (Developer Environment)
- ติดตั้งโปรแกรมที่จำเป็นสำหรับการพัฒนา เช่น Visual Studio Code, Node.js และ Git
- สร้างบัญชี GitHub และสร้าง Repository สำหรับจัดเก็บโครงการ
- ฝึกใช้งาน Git เบื้องต้นในการจัดการเวอร์ชันของซอร์สโค้ด
- เชื่อมต่อ Git กับ GitHub และอัปโหลดโปรเจกต์ขึ้น Repository
- เรียนรู้โครงสร้างของโปรเจกต์และการจัดทำเอกสาร README.md
